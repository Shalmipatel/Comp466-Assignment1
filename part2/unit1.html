Web

Introduction

A web server responds to client requests by providing resources such as HTML documents.

HTTP

URI: (Uniform Resource Identifiers) identify resources on the internet.

URL: Starts with "http://" and refers to files, directories or server-side code that performs tasks, such as database lookups, Internet searches and business-application processing.

How do http transactions work?

  1. The request is send from the web client to the web server
  2. After it recives the request, the web server seraches its system for the resources
  3. the server responds to the request with resource's content

Two most common HTTP request types are:

  • Get
  • Post
